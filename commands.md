## Commands

#### To initialize git into your project
```
git init
```


#### To check status
```
git status
```

#### To add files
```
git add <<filename>>

OR to add all files

git add .
```

#### To commit
```
git commit -m "Commit message"
```

### To check current branch
```
git branch
```

### to add remote
```
git remote add origin https://github.com/zashishz/team-records.git
```

### to push changes
```
git push origin master
```
